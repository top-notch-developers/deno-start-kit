Run these commands to get started

## Run the program
```
deno run main.ts
```

- Allow net
```
deno run --allow-net=deno.com [file_name]
```

## Run the program and watch for file changes
```
deno task dev
deno run --watch [file_name]
```

## Run the tests
```
deno test
```

## Run the benchmarks
```
deno bench
```
